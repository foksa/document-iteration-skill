# JetBrains IDEs

Visual highlighting of iteration markers in IntelliJ IDEA, WebStorm, PyCharm, and other JetBrains IDEs.

## TODO Patterns

Settings → Editor → TODO:

Add patterns:
- `\%\%.*\%\%` - Comments
- `>>.*>>` - Notes
- `==.*==\(.*\)` - Highlights
